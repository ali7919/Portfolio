Online Demo

  • Category: Unity
  • Start Date: January 2022 (Bahman 1400)
  • Project URL: Github

I used the third person template provided by unity and turned it into an online game using PUN2 (photon #unity networking). The way it works is simple, someone becomes the host, and others can join, I also coded a prize spawning-collecting system that is operated by the host.

در این پروژه من third person template یونیتی که شامل محیط و کارکتر است رو با استفاده از photon تبدیل به یک بازی آنلاین کردم همچنین یک سیستم اسپان و دریافت جایزه (توپ های زرد) رو پیاده سازی کردم که آنلاین و سینک هست و مدیریتش با host میباشد. و علاوه بر این یک انیمیشن رقص هم به سیستم انیمیشن اضافه کردم.
کل این پروژه open source است و میتونین اونو در گیتهاب من مشاهده کنید
